# **FlightApp : Reactive Flight Booking System (WebFlux + Reactive MongoDB)**

**This project**
A Reactive Spring WebFlux application for managing flights and bookings. It supports adding flights, searching flights, booking seats, and cancelling bookings. The system is fully non-blocking and uses MongoDB as the database.

**What This Project Includes**

* Reactive WebFlux (fully non-blocking)
* MongoDB for storage

**Endpoints for**

* Add flights
* Search flights
* Book tickets
* Cancel bookings

**Additional Features**

* Global exception handling
* DTO-based request/response structure
* Validation on input fields
* SonarQube + SonarCloud code quality checks
* Jacoco test coverage (~80%)
* JMeter load tests (1, 10, 50, 100 threads)
